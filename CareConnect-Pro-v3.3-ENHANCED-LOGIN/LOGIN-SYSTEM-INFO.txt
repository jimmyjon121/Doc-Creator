ACCOUNT MANAGEMENT:
-------------------
- Accounts are stored in browser's localStorage
- Each device maintains its own account database
- Multiple accounts can exist on the same device
- Last login time is tracked for each account

To Clear All Accounts (Reset):
1. Open browser developer tools (F12)
2. Go to Application/Storage tab
3. Find localStorage for the app
4. Delete 'careconnect_user_accounts' key
5. Refresh the page
