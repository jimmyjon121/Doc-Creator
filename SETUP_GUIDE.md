# FFAS Document Creator - Setup Guide
## By ClearHive Health LLC

This is a **completely self-contained** HTML application that runs directly in your browser. No server or installation required!

## Quick Start (2 Minutes)

### Step 1: Open the Application
Simply double-click `AppsCode.html` or open it in any modern web browser (Chrome, Firefox, Safari, Edge).

### Step 2: Configure AI (One-Time Setup)
When you first try to add a custom program using AI:

1. **Choose your AI provider:**
   - **OpenAI** (ChatGPT) - Recommended for most users
   - **Anthropic** (Claude) - Alternative option

2. **Get your API key:**
   - **For OpenAI:** Visit https://platform.openai.com/api-keys
     - Sign up/login to OpenAI
     - Click "Create new secret key"
     - Copy the key (starts with `sk-`)
   
   - **For Anthropic:** Visit https://console.anthropic.com/account/keys
     - Sign up/login to Anthropic
     - Create a new API key
     - Copy the key (starts with `sk-ant-`)

3. **Enter your key:**
   - Paste your API key when prompted
   - Click "Save & Continue"
   - Your key is saved locally in your browser (never shared)

## Troubleshooting

### "AI Not Configured" Message
- You need to add an API key (see Step 2 above)
- Keys are saved after first entry

### Programs Not Showing
- Clear browser cache and reload
- Check browser console for errors (F12)

### Document Generation Issues
- Ensure at least one program is selected
- Enter client name before generating

### API Errors
- Verify your API key is correct
- Check you have credits in your API account
- Try the alternative API provider
